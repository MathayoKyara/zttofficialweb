## Image Requirements

### Tour Images
- **Size**: 1200x800 pixels (16:9 aspect ratio)
- **Format**: JPEG
- **Quality**: High quality, optimized for web
- **Content**: Should showcase the main attraction or activity

### Hidden Gems Images
- **Size**: 1200x800 pixels (16:9 aspect ratio)
- **Format**: JPEG
- **Quality**: High quality, optimized for web
- **Content**: Should showcase the unique aspect of each hidden gem

### Pattern Image
- **Size**: 512x512 pixels
- **Format**: PNG with transparency
- **Content**: Subtle pattern for background decoration

## Image Sources

For production use, you should source high-quality images from:
1. Professional photographers
2. Stock photo services (Shutterstock, iStock, etc.)
3. Local Zanzibar photographers
4. Your own photography

## Placeholder Images

Until you have the actual images, the app will display placeholder icons. To add real images:

1. Create the directory structure above
2. Add your images with the exact filenames listed
3. Ensure images are optimized for mobile devices
4. Test the app to ensure all images load correctly

## Image Optimization Tips

- Compress images to reduce file size while maintaining quality
- Use WebP format for better compression (with JPEG fallback)
- Consider different sizes for different screen densities
- Test loading times on slower connections
