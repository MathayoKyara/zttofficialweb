# Images Directory

This directory should contain the following images for the Zanzibar Trail Tours application:

## Required Images

### Hero Section
- `hero_bg.jpg` - Main hero background image (recommended: 1920x1080 or higher)

### Tour Images
- `spice_tour.jpg` - Zanzibar spice tour
- `serengeti.jpg` - Serengeti safari
- `stone_town.jpg` - Stone Town exploration
- `prison_island.jpg` - Prison Island visit
- `nungwi.jpg` - Nungwi & Kendwa beaches
- `jozani.jpg` - Jozani Forest
- `culture.jpg` - Cultural experiences
- `ngorongoro.jpg` - Ngorongoro Crater
- `kilimanjaro.jpg` - Mount Kilimanjaro
- `selous.jpg` - Selous Game Reserve
- `dar_es_salaam.jpg` - Dar es Salaam
- `lake_victoria.jpg` - Lake Victoria

### Gallery Images
- `gallery1.jpg` through `gallery12.jpg` - Various tour and destination images

### Testimonial Images
- `testimonial1.jpg` through `testimonial5.jpg` - Customer profile images

### About Section
- `about_us.jpg` - Company/about us image

## Image Specifications

- **Format**: JPG or PNG
- **Resolution**: Minimum 800x600, recommended 1920x1080 or higher
- **File Size**: Optimize for web/mobile (under 500KB per image)
- **Aspect Ratio**: 16:9 or 4:3 recommended for consistency

## Placeholder Images

- https://picsum.photos/ for random images
